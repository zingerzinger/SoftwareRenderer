# SoftwareRenderer

SoftwareRenderer is a simple game engine that inludes:

- resource loader
- software renderer
- physics engine
- sound subsystem

<b>Build & Run:</b> <br>

1. Install [.Net SDK 9](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
2. ```> dotnet build```
3. ```> cd build```
4. ```> SoftwareRenderer.exe```
